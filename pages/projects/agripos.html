AGRI-POS: Digital Manajemen Retail Pertanian

AGRI-POS adalah solusi digital untuk membantu pelaku ritel pertanian dalam mengelola transaksi, stok produk, dan distribusi data penjualan secara terpadu dalam satu sistem.

Desain antarmuka dibuat agar staf operasional dapat bekerja lebih cepat, meminimalkan kesalahan pencatatan, dan memudahkan pengambilan keputusan berbasis data.

Lihat Demo Github

Technologies Used

Figma HTML5 CSS3 JavaScript

Key Features

  1. Pencatatan transaksi harian untuk pembelian, penjualan, dan retur secara real-time.
  2. Monitoring stok produk dengan ringkasan kategori dan notifikasi kebutuhan restok.
  3. Laporan performa penjualan untuk membantu evaluasi operasional dan strategi bisnis.
