Website Kedai Kopi

Proyek ini menampilkan branding dan menu kedai kopi melalui desain web yang hangat, visual kuat, dan storytelling produk yang menarik.

Struktur halaman dibuat agar pengunjung mudah menemukan informasi menu, promo, serta profil kedai, sekaligus mendorong interaksi dan pemesanan.

Lihat Demo Github

Technologies Used

Figma HTML5 CSS3 JavaScript

Key Features

  1. Hero section visual untuk memperkuat identitas brand kedai kopi.
  2. Daftar menu terstruktur dengan fokus pada keterbacaan dan estetika produk.
  3. Tampilan responsif agar pengalaman pengguna tetap nyaman di semua perangkat.
