E-Commerce Flower Shop

Inforlist Home adalah platform belanja bunga online yang dirancang untuk memberikan pengalaman berbelanja yang estetik dan praktis. Proyek ini fokus pada kemudahan navigasi katalog produk, mulai dari bunga segar hingga kado custom.

Dengan antarmuka modern, pengguna dapat dengan mudah memilih rangkaian bunga terbaik untuk menyampaikan perasaan mereka di momen-momen spesial.

Lihat Demo Github

Technologies Used

Figma HTML5 CSS3 JavaScript

Key Features

  1. Katalog produk estetik dengan kategori rangkaian bunga, buket uang, dan kado custom.
  2. Sistem pemesanan yang mudah untuk memilih produk, menentukan pesanan, dan pengiriman.
  3. Desain responsif agar nyaman digunakan di desktop maupun smartphone.
