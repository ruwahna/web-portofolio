UI/UX Sistem Kasir Posfy

POSY adalah sistem kasir digital untuk membantu operasional bisnis ritel, mulai dari transaksi penjualan, manajemen item, hingga ringkasan performa harian.

Antarmuka dirancang dengan pendekatan efisien agar kasir dapat bekerja lebih cepat, mengurangi kesalahan input, dan tetap nyaman digunakan dalam jam operasional yang padat.

Lihat Desain Dribbble

Technologies Used

Figma HTML5 CSS3 JavaScript

Key Features

  1. Dashboard ringkas untuk memantau transaksi dan status operasional harian.
  2. Alur kasir cepat untuk pencarian produk, input jumlah, dan penyelesaian pembayaran.
  3. Ringkasan performa penjualan untuk mendukung evaluasi bisnis secara berkala.
